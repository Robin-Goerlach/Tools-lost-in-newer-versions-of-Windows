NtClock - clean WinForms project without Designer files

Open NtClock.sln in Visual Studio 2022.
This project uses only code-created controls, so there are no .Designer.cs or .resx files to conflict.

Build requirements:
- Visual Studio 2022 Community is fine
- Workload: .NET desktop development
- Target framework: .NET 8 Windows

Important:
- If you already tried older versions, create a NEW empty folder and use only the files from this ZIP.
- Do not mix these files with older ClockForm.cs / SettingsForm.cs / Designer files.

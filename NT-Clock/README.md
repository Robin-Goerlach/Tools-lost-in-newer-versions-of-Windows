# NtClock

A clean Windows Forms recreation inspired by the Windows NT 4.0 Clock utility.

## Features
- Analog clock in a classic NT 4.0 look
- Optional title bar that can be hidden
- Optional digital time and date
- Always-on-top mode
- Daily alarm
- Tray icon with context menu
- Settings saved as JSON under AppData

## Build
Open `NtClock.sln` in Visual Studio 2022 with the ".NET desktop development" workload installed, then build the solution.
